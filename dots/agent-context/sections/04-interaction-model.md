## Interaction model

- If a user request is unclear, ask clarifying questions until the execution
  steps are perfectly clear.
- Once clarified, proceed autonomously without asking for human intervention.
- Ask for help only when: a script runs longer than 2 minutes (use timeout,
  then ask the user to run it manually), elevated privileges are required
  (sudo), or another critical blocker cannot be resolved programmatically.
